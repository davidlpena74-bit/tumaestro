// turbo-all

Este agente utiliza IA para preparar los assets visuales de los juegos, eliminando fondos automáticamente.

1. **Configuración**: El script `web/scripts/images_specialist_agent.mjs` debe contener las rutas de origen y destino.
2. **Ejecución**: Ejecutar el script procesador. `node scripts/images_specialist_agent.mjs` (desde el directorio `web`).
3. **Resultado**: Las imágenes optimizadas se guardan en `web/public/images/games/`.
